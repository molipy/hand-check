# hand-check
基于深度学习的【手势识别】系统实现~2026原创+计算机毕设

## 项目介绍

本项目是一个基于深度学习的静态数字手势识别系统，主要用于精准识别包含数字0至9的十种常见手势。系统采用前后端分离的现代化Web架构，前端基于Vue3框架结合Element Plus组件库构建，为用户提供直观、响应迅速且友好的交互界面，支持手势图片的上传、实时结果展示及历史记录查询；后端采用轻量级Web框架Flask，负责业务逻辑处理、API接口提供、用户权限校验（JWT）以及与数据库的交互；核心算法模块则依托TensorFlow深度学习框架，采用经典的ResNet50（残差网络）作为主干网络，对采集到的手势图像进行深层特征提取与分类识别。
